# Mini E-Learning Platform

## Project Overview
This is a **mini e-learning platform prototype** built using **HTML5, CSS3, and JavaScript**.  
The project demonstrates the ability to build a functional, interactive website using vibe coding concepts, focusing on a clean structure and interactivity.

**Purpose:**  
To provide learners with a small platform to view courses, see details, and mark courses as completed.

---

## Pages Included
1. **Home (`index.html`)** – Displays a list of available courses.  
2. **Course Details (`course.html`)** – Shows course information and allows marking a course as completed.  
3. **Login (`login.html`)** – Optional login/sign-up page for future extension.

---

## Features
- Responsive layout across desktop and mobile devices  
- Navigation between pages with consistent header and footer  
- Interactive elements:
  - Click to view course details
  - Mark a course as completed (with JS feedback)
- Clean, modular, and readable code
